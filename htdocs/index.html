Services: nshmp-haz-ws

Hazard Curve

A USGS hazard curve may be computed by supplying a slash-delimited
or name-value pair query consisting of edition, region, longitude,
latitude, intensity measure type (IMT), and Vs30 value. For example:

http://.../nshmp-haz-ws/hazard/E2008/WUS/-118.25/34.05/PGA/760

    – or –

http://.../nshmp-haz-ws/hazard?edition=E2008&region=WUS&longitude=-118.25&latitude=34.05&imt=PGA&vs30=760

  – where –

edition = [E2008, E2014]
region = [COUS, WUS, CEUS]
vs30 = [180, 259, 360, 537, 1150, 2000]
imt = [PGA, SA0P2, SA1P0]
For both forms, multiple, comma-delimited IMTs may be supplied; for
the name-value pair format, IMT may be omitted and the service will
return curves for all supported IMTs.

See usage for parameter dependencies.

Deaggregation

A USGS deaggregation may be computed by supplying a slash-delimited
or name-value pair query consisting of edition, region, longitude,
latitude, intensity measure type (IMT), Vs30 value and return period. For example:

http://.../nshmp-haz-ws/deagg/E2008/WUS/-118.25/34.05/PGA/760/2475

    – or –

http://.../nshmp-haz-ws/deagg?edition=E2008&region=WUS&longitude=-118.25&latitude=34.05&imt=PGA&vs30=760&returnPeriod=2475

  – where –

edition = [E2008, E2014]
region = [COUS, WUS, CEUS]
vs30 = [180, 259, 360, 537, 1150, 2000]
imt = [PGA, SA0P2, SA1P0]
returnPeriod = [1, 5,000]

See usage for parameter dependencies.
